еүҚиЁҖ

Java I/OпјҲInput/OutputпјүжөҒжҳҜJavaзЁӢеәҸдёҺеӨ–йғЁдё–з•ҢиҝӣиЎҢж•°жҚ®дәӨжҚўзҡ„йҮҚиҰҒжЎҘжўҒгҖӮж— и®әжҳҜиҜ»еҸ–ж–Үд»¶гҖҒзҪ‘з»ңйҖҡдҝЎпјҢиҝҳжҳҜжҺ§еҲ¶еҸ°дәӨдә’пјҢйғҪзҰ»дёҚејҖI/OжөҒзҡ„ж”ҜжҢҒгҖӮJavaжҸҗдҫӣдәҶдё°еҜҢзҡ„I/Oзұ»еә“пјҢж”ҜжҢҒеӯ—иҠӮжөҒгҖҒеӯ—з¬ҰжөҒгҖҒзј“еҶІжөҒгҖҒеҜ№иұЎжөҒзӯүеӨҡз§Қж•°жҚ®дј иҫ“ж–№ејҸгҖӮжң¬ж–Үе°Ҷж·ұе…ҘжҺўи®ЁJava I/OжөҒзҡ„ж ёеҝғжҰӮеҝөгҖҒеҲҶзұ»дҪ“зі»гҖҒдҪҝз”Ёж–№жі•д»ҘеҸҠе®һйҷ…еә”з”ЁеңәжҷҜгҖӮ

I/OжөҒзҡ„еҹәжң¬жҰӮеҝө

I/OжөҒжҳҜJavaдёӯеӨ„зҗҶиҫ“е…Ҙиҫ“еҮәзҡ„жҠҪиұЎжҰӮеҝөпјҢе®ғе°Ҷж•°жҚ®зҡ„дј иҫ“жҠҪиұЎдёәжөҒзҡ„еҪўејҸпјҢдҪҝеҫ—зЁӢеәҸеҸҜд»Ҙз”Ёз»ҹдёҖзҡ„ж–№ејҸеӨ„зҗҶдёҚеҗҢжқҘжәҗе’Ңзӣ®ж Үзҡ„ж•°жҚ®гҖӮ

I/OжөҒзҡ„еұӮж¬Ўз»“жһ„

Java I/OжөҒйҮҮз”ЁиЈ…йҘ°еҷЁжЁЎејҸи®ҫи®ЎпјҢеҪўжҲҗдәҶе®Ңж•ҙзҡ„зұ»еұӮж¬Ўз»“жһ„пјҡ

еӯ—иҠӮжөҒеұӮж¬Ўз»“жһ„

еӯ—з¬ҰжөҒеұӮж¬Ўз»“жһ„

еӯ—иҠӮжөҒиҜҰи§Ј

еӯ—иҠӮжөҒд»Ҙеӯ—иҠӮдёәеҚ•дҪҚиҝӣиЎҢж•°жҚ®дј иҫ“пјҢйҖӮз”ЁдәҺеӨ„зҗҶдәҢиҝӣеҲ¶ж•°жҚ®пјҢеҰӮеӣҫзүҮгҖҒйҹійў‘гҖҒи§Ҷйў‘зӯүж–Үд»¶гҖӮ

InputStreamе’ҢOutputStream

ж–Үд»¶еӯ—иҠӮжөҒеҹәжң¬ж“ҚдҪң

ByteArrayInputStreamе’ҢByteArrayOutputStream

еӯ—з¬ҰжөҒиҜҰи§Ј

еӯ—з¬ҰжөҒдё“й—Ёз”ЁдәҺеӨ„зҗҶж–Үжң¬ж•°жҚ®пјҢиҮӘеҠЁеӨ„зҗҶеӯ—з¬Ұзј–з ҒиҪ¬жҚўпјҢйҖӮз”ЁдәҺеӨ„зҗҶж–Үжң¬ж–Үд»¶гҖӮ

е®һйҷ…еә”з”ЁжЎҲдҫӢ

жҖ»з»“

Java I/OжөҒжҳҜJavaзј–зЁӢдёӯзҡ„йҮҚиҰҒз»„жҲҗйғЁеҲҶпјҢдёәзЁӢеәҸжҸҗдҫӣдәҶејәеӨ§зҡ„ж•°жҚ®еӨ„зҗҶиғҪеҠӣгҖӮйҖҡиҝҮжң¬ж–Үзҡ„еӯҰд№ пјҢжҲ‘д»¬ж·ұе…ҘдәҶи§ЈдәҶпјҡ

  1. I/OжөҒдҪ“зі»пјҡжҺҢжҸЎдәҶеӯ—иҠӮжөҒгҖҒеӯ—з¬ҰжөҒзҡ„еҲҶзұ»е’ҢеұӮж¬Ўз»“жһ„
  2. еҹәзЎҖжөҒж“ҚдҪңпјҡеӯҰдјҡдәҶж–Үд»¶иҜ»еҶҷгҖҒж•°жҚ®дј иҫ“зҡ„еҹәжң¬ж–№жі•
  3. зј“еҶІжөҒдјҳеҢ–пјҡзҗҶи§ЈдәҶзј“еҶІжңәеҲ¶еҜ№жҖ§иғҪзҡ„йҮҚиҰҒеҪұе“Қ
  4. ж•°жҚ®жөҒеә”з”ЁпјҡжҺҢжҸЎдәҶеҹәжң¬ж•°жҚ®зұ»еһӢе’ҢеҜ№иұЎзҡ„еәҸеҲ—еҢ–
  5. е®һз”ЁжҠҖе·§пјҡеӯҰдјҡдәҶж–Үд»¶е·Ҙе…·зұ»зҡ„и®ҫи®Ўе’ҢжҖ§иғҪдјҳеҢ–ж–№жі•

еңЁе®һйҷ…ејҖеҸ‘дёӯпјҢжӯЈзЎ®дҪҝз”ЁI/OжөҒиғҪеӨҹпјҡ

  • жҸҗй«ҳзЁӢеәҸзҡ„ж•°жҚ®еӨ„зҗҶиғҪеҠӣ
  • дјҳеҢ–ж–Үд»¶ж“ҚдҪңжҖ§иғҪ
  • е®һзҺ°еӨҚжқӮзҡ„ж•°жҚ®дәӨжҚўйңҖжұӮ
  • жһ„е»әеҒҘеЈ®зҡ„ж–Үд»¶еӨ„зҗҶзі»з»ҹ

е»әи®®ејҖеҸ‘иҖ…еңЁдҪҝз”ЁI/OжөҒж—¶жіЁж„Ҹпјҡ

  • еҸҠж—¶е…ій—ӯжөҒиө„жәҗпјҢжҺЁиҚҗдҪҝз”Ёtry-with-resources
  • ж №жҚ®ж•°жҚ®зұ»еһӢйҖүжӢ©еҗҲйҖӮзҡ„жөҒ
  • дҪҝз”Ёзј“еҶІжөҒжҸҗй«ҳжҖ§иғҪ
  • жіЁж„Ҹеӯ—з¬Ұзј–з Ғй—®йўҳ
  • еҰҘе–„еӨ„зҗҶејӮеёёжғ…еҶө

жҺҢжҸЎJava I/OжөҒзҡ„зІҫй«“пјҢе°Ҷдёәжһ„е»әй«ҳж•ҲгҖҒеҸҜйқ зҡ„Javaеә”з”ЁзЁӢеәҸеҘ е®ҡеқҡе®һеҹәзЎҖгҖӮ
