📑 Junly文件转换工具

📝 产品简介

Junly文件转换工具是一款功能强大的多格式文档转换软件，支持Word、Excel、PDF、Markdown等多种格式的互相转换，同时提供图片处理和文本处理功能。软件界面简洁直观，操作便捷，是日常文档处理的得力助手。

🔗 开源信息

🌟
项目已开源！ 欢迎参与贡献、提出建议或报告问题。
GitHub地址：https://github.com/wujunbiao1002/file-converter

⭐ 开源优势

  • 自由使用与修改：可根据个人或组织需求定制功能
  • 社区贡献：集合多人智慧，持续改进产品
  • 透明度：代码公开，安全可靠，无后门
  • 学习参考：对Python开发和GUI设计感兴趣的开发者可作为学习资源

💻 技术栈

  • 前端界面：PyQt6、Qt Designer
  • 核心处理：Python 3.8+
  • 依赖库：python-docx, openpyxl, PyPDF2, Pillow等
  • 构建工具：PyInstaller

🔍 核心功能

📄 Word转换功能
  • 支持Word转Markdown/HTML，保留图片和格式
  • 支持单个和批量操作
🖼️ Word去除图片转换功能
  • 支持去除Word中的图片并转换为多种格式
📊 Excel转换功能
  • 支持Excel转TXT/Markdown
📝 Markdown表格数据转Excel功能
  • 自动提取表格数据并按表头分组
📄 Markdown转Word功能
  • 保留格式转换为Word文档
📑 PDF转换功能
  • 支持PDF转Word/TXT及图片提取
🖼️ 图片处理功能
  • 批量格式转换、尺寸调整和压缩
📝 文本处理功能
  • 编码转换、格式转换和批量替换

🖥️ 用户界面

  • 标签页形式组织不同功能模块
  • 支持文件拖拽操作
